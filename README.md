A comprehensive, easy-to-use web application designed specifically for the students of **BGMEA University of Fashion & Technology (BUFT)**. 

This hub allows students to instantly generate perfectly formatted, print-ready academic cover pages and experiment indexes, completely eliminating the hassle of formatting margins and borders in Microsoft Word.

**🌐 Live Website:** [https://bufthub.vercel.app](https://bufthub.vercel.app)

---

## ✨ Key Features

*   **📝 Assignment Cover Generator:** Create premium, high-contrast assignment layouts with customizable title decorations and teacher-student department grids.
*   **🔬 Lab Report Cover Generator:** Build fully structured laboratory experiment reports with precise experiment numbers, performance dates, and standardized department titles.
*   **📊 Index Page Creator:** Assemble high-fidelity laboratory index registers. Automatically calculates experiment counts and lays out perfect cells for grading and teacher signatures.
*   **🧮 CGPA Calculator:** Plan and project your academic grades! Calculate semester GPAs, track multiple sessions, and project target CGPAs based on the official BUFT grading scale.
*   **🖨️ High-Resolution PDF Export:** Generates crystal-clear, exact A4-sized (210mm x 297mm) PDFs with pixel-perfect double-border alignments.
*   **🚌 BUFT Bus Tracker Integration:** Quick access to the real-time BUFT shuttle tracker to help students manage their daily commutes.

## 🚀 Built With

This project is built using modern front-end web technologies to ensure a fast, responsive, and lightweight experience:

*   **[React 19](https://react.dev/)** - UI Library
*   **[TypeScript](https://www.typescriptlang.org/)** - Type Safety
*   **[Vite](https://vitejs.dev/)** - Build Tool & Development Server
*   **[Tailwind CSS v4](https://tailwindcss.com/)** - Styling and Layout
*   **[jsPDF](https://github.com/parallax/jsPDF) & [html2canvas](https://html2canvas.hertzen.com/)** - Client-side PDF generation
